# MIRU2026 若手プログラム — ベンチマーク論文をベンチマークする

MIRU2026若手プログラム「わかりやすい、読みやすいを深掘る」のベンチマークグループによるWebサイトです。調査から得たベンチマーク論文の読みやすさに関する指針と、サーベイ対象の論文一覧を公開します。

## ページ仕様

サイトはビルド工程を必要としない、GitHub Pages向けの静的な単一ページです。

### コンテンツ構成

1. プロジェクト概要
2. ベンチマークを提案する4つのモチベーション
   - 領域開拓型
   - 欠落補完型
   - 更新・難化型
   - 評価再設計型
3. わかりやすいベンチマーク論文を構成する7つの観点
4. 調査論文リスト
5. プロジェクトメンバーと発表ポスター

### 論文リスト

論文データは `static/js/papers.js` から読み込みます。現在の主な機能は以下のとおりです。

- タイトル・著者・掲載先・分野・コメントを対象としたキーワード検索
- 分野による絞り込み
- 各論文のモチベーション分類を色付きラベルで表示
- 掲載順、新しい順、古い順、タイトル順での並び替え
- 論文のポイントと「わかりやすさ」コメントの展開表示
- 外部論文ページへのリンク
- 16件単位の追加表示
- `Command/Ctrl + K` による検索欄へのフォーカス

### デザイン

- テーマカラー：`#072F90`
- 背景色：`#F7F9FF`
- 日本語フォント：Noto Sans JP
- 欧文フォント：DM Sans
- 760px以下でモバイルレイアウトに切り替え

4つのモチベーションには次のカラーパレットを使用します。

| 分類 | 背景色 | アクセント色 |
| --- | --- | --- |
| 領域開拓型 | `#FEF1F0` | `#C82E40` |
| 欠落補完型 | `#FEF5E6` | `#D56F03` |
| 更新・難化型 | `#F2FFF5` | `#17803B` |
| 評価再設計型 | `#F3F7FF` | `#3267BA` |

## 論文データの更新

`static/data/サーベイ論文（ベンチマークグループ）.xlsx` の「サーベイ論文リスト」シートを編集後、次のコマンドをリポジトリルートで実行します。

```bash
python3 scripts/extract_papers.py \
  'static/data/サーベイ論文（ベンチマークグループ）.xlsx' \
  static/js/papers.js
```

抽出スクリプトはPython標準ライブラリのみを使用します。Excelの列は次のように変換されます。

| Excel列 | 出力フィールド | 用途 |
| --- | --- | --- |
| A | `url` | DOIまたは論文URL |
| D | `title` | 論文名 |
| E | `authors` | 著者 |
| F | `venue` | 掲載ジャーナル・学会 |
| G | `year` | 掲載年月日 |
| H | `field` | 分野・ジャンル |
| I | `motivation` | モチベーション分類 |
| J | `clarity` | わかりやすさに関するコメント |
| K | `summary` | 論文のポイント・紹介理由 |

生成された `static/js/papers.js` もGitへ追加してください。内容は自動生成されるため、原則として直接編集しません。

## ローカル確認

相対リンクを本番と同じ条件で確認する場合は、リポジトリルートでローカルサーバーを起動します。

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000/` を開いてください。

## 公開時の確認事項

- `static/data/` のPDFとExcelがGitに追加されていること
- Excel更新後に `static/js/papers.js` を再生成していること
- ポスターリンクと各論文の外部リンクが開けること
- デスクトップとモバイルの両方で表示を確認すること
